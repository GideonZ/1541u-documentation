Preface
=======

This edition consolidates the foundational UCI register interface and
   the currently documented targets into a single manual with a
   consistent structure, chapter ladder, and terminology. The goal is to
   preserve the technical intent of the original documents while making
   the material easier to read sequentially, browse by target, and use
   as a long-lived programming reference.

Editorial note: legacy source documents use slightly different naming
conventions (for example “Ultimate-II command interface”, “UCI”, and
target-specific labels). In this unified edition, chapter titles and
running terminology have been harmonized where possible, while command
names, register names, and behavior descriptions remain technically
unchanged.
